# Audiokinetic Wwise SDK (2008) v2.1.2821
Audiokinetic Wwise SDK 2008 v2.1.2821

This repository contains **Wwise 2008 (version 2.1.2821)**, the audio middleware engine from Audiokinetic. Wwise is widely used in games and interactive applications for **real-time audio integration, sound management, and interactive music**.

> This repository may contain ancient source code, headers, and examples. It does **not** include modern Wwise tools or installers. Intended for archival, research, or legacy project use.

## Purpose of this Repository

This repo is a reference code for software preservationists. This is published here to provide public access to ones wanting to dig into Wwise-powered games and applications of early 2000s. It peserves the **source code of the Wwise 2008, version 2.1.2821** for:

- Legacy project maintenance  
- Audio middleware research and experimentation  
- Educational or archival purposes

## What's Included

- **Source Code / SDK**  
  The core Wwise engine code and API for integration.

- **Header Files**  
  Necessary headers for building or interfacing with the engine.

- **Examples**  
  Sample projects demonstrating basic audio integration and API usage.

- **Documentation**  
  Original API reference and technical notes from the 2008 release.

## Folder Structure

/src
Wwise engine source code
/include
Header files for API access
/samples
Example projects demonstrating audio integration
/docs
Documentation and API reference

## Getting Started

1. Include the `include/` directory in your project.  
2. Build the source code or link against the provided libraries (if any).  
3. Run the sample projects in `samples/` to verify setup.  
4. Consult `docs/` for API reference and integration notes.
